# 🌱 EcoUninorte — Plataforma de Gobernanza Ambiental

Plataforma web interactiva para fortalecer la gobernanza institucional frente al manejo de residuos en la **Universidad del Norte**, Barranquilla, Colombia.

## Inicio Rápido

```bash
npm install
npm run dev
```

El proyecto se ejecutará en: `http://localhost:5173`

## Stack Tecnológico

- **React** + **Vite**
- **CSS Puro** (Vanilla CSS) — Sin Tailwind
- **Framer Motion** — Animaciones fluidas

## Conceptos Teóricos Integrados

1. **Antropoceno** — Era actual donde la actividad humana es la fuerza dominante del planeta
2. **Cambio Global** — Alteraciones ambientales por actividades humanas
3. **Límites Planetarios** — 9 umbrales que no debemos cruzar
4. **Servicios Ecosistémicos** — Beneficios que los ecosistemas brindan
5. **Ciencia Ciudadana** — Participación de todos en la generación de datos ambientales

## Funcionalidades

- Landing page con estadísticas y animaciones
- Explicación del problema central del campus
- Tarjetas interactivas de conceptos ecológicos
- Propuesta de gobernanza institucional participativa
- Módulo de Ciencia Ciudadana (reportes simulados)
- Minijuego de clasificación de residuos (Drag & Drop)
- Sección de cierre motivacional
- Cursor personalizado con glow
- Partículas ambientales de fondo
- Diseño 100% responsive
